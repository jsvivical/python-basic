cd / chdir  + 경로 : 해당하는 디렉토리로 진입
cd.. : 상위 디렉터리로 바꿈

cls : 화면에 표시된 것을 모두 지움

color : 콘솔의 기본색과 배경색 설정

copy : 하나 이상의 파일을 다른 위치로 복사
copy [[/D] [/V] [/N] [/Y | -/Y] [/Z] [/L] [/A | /B] 원본파일] + 대상[/A | /B] 


date : 날짜를 보여주거나 설정
dir : 디렉터리에 있는 파일과 하위 디렉토리 목록을 보여줌
findstr : 파일에서 문자열을 찾음
help : 윈도우즈 명령어에 도움말 제공
md : 디렉토리를 만듦
mkdir : 디렉토리를 만듦
mode : 시스템 장치 구성
rd : 디렉토리를 지움
ren : 파일이름을 바꿈
rename : 파일이름을 바꿈
redir : 디렉토리를 지움
set : 윈도우즈 환경변수를 출력, 설정, 지움
start : 지정된 프로그램이나 명령을 별도의 창에서 실행
tree : 드라이브 또는 경로의 디렉터리 구조를 그래픽으로 화면에 표시된
type : 텍스트 파일의 내용을 보여줌
